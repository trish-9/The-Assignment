## ▶️ Running the Application

### 🖥️ Run the CLI Program

To run the command-line version of the reflection agent:

1. Open your terminal
2. Navigate to the project directory
3. Run the following command:

```bash
python agent.py
```

This will start the interactive reflection session in your terminal.

---

### 🌐 Run the Web Application

To run the web-based version:

1. Open the project folder in **VS Code**
2. Install the **Live Server extension** (if not already installed)
3. Right-click on `index.html`
4. Click **"Open with Live Server"**

This will launch the reflection app in your browser.

---

### 💡 Notes

* Make sure `reflection-tree.json` is in the same directory as your program
* The web version dynamically loads the tree file — no hardcoding is used
* Both CLI and Web versions follow the same deterministic logic
